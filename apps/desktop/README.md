# Desktop App

Electron desktop UI for the `key-b0x` runtime.

Stack:

- Electron
- React
- TypeScript
- Vite

Responsibilities:

- edit normalized key bindings
- install/check the Slippi profile
- start and stop the Rust runtime
- guide the user through a simple wizard-style flow

Useful commands:

- `npm install`
- `npm run dev`
- `npm run typecheck`
- `npm run test`
- `npm run build`
