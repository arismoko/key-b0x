# Desktop App

Tauri desktop UI for `key-b0x`.

Stack:

- Tauri v2
- React
- TypeScript
- Vite

Responsibilities:

- edit normalized key bindings
- install/check the Slippi profile
- start and stop the in-process runtime controller
- guide the user through a simple wizard-style flow

Useful commands:

- `npm install`
- `npm run dev`
- `npm run typecheck`
- `npm run test`
- `npm run build`
